Natural Language Processing in the Cloud

01 Feb 2015

In recent years cloud-based IDEs have become increasingly popular. Services like Koding and Cloud9 provide users with web-based development environments that make it easy to access your projects wherever you are and even collaborate with other users in real-time. The two previously mentioned services provide users with their own virtual machine (VM) running Ubuntu, and are preconfigured with Python, Node.js, Ruby, PHP, and more so that you can immediately jump in and start being productive.

NLTK, The Natural Language Toolkit, is a collection of natural language processing libraries and programs for Python. I’ll show you how simple it is to install NLTK on your VM so you can get started learning about natural language processing without having to worry about configuring it on multiple devices. In addition, I’ll walk through installing the NLTK Data packages used in Natural Language Processing with Python so that you can follow along with this great resource.

We’ll be using Koding because that’s the service I’m more familiar with and because it provides the most free space out of all of the cloud IDEs that I’ve investigated, which leads us into an important factor to note: Koding’s free accounts are limited to 3 gigs of disk space. While this is enough to get started with NLTK and download the data libraries that we’ll use to follow along with the book, you may run into problems trying to install additional packages down the road.

1. Setup your Koding account

To begin, you’ll have to set up your Koding account. Head on over to their site, register, and complete the email confirmation process. Once this is complete you should be able to log into your workspace. The first time you log in you will be prompted to initialize your VM. This process may take a minute or two.

Take a moment to acquaint yourself with the Koding interface. On the left-hand side you’ll find menus, settings, and the file tree for your workspace. On the right side you’ll see the text editor and below that is the terminal. We’ll be working in the terminal to get NLTK installed.

2. Install pip

pip is a package manager for Python that will make it extremely easy to install NLTK and other Python packages. pip does not come pre-installed on our VM, so we’ll have to go down to our terminal and enter:

sudo apt-get install python-pip

You’ll be asked if you want to continue. Type Y and press Enter.


3. Install NLTK

Now that pip has been installed we can install nltk with a single command in our terminal. Go ahead and enter:

sudo pip install -U nltk

As the install runs you’ll probably receive a few warnings. You can ignore these.

To test that NLTK has been properly installed, enter python in your terminal. Once Python is running, enter:

import nltk

If no errors are returned, NLTK has been succesfully installed. You should see something like this.


4. Download NLTK Data packages

NLTK comes with a number of data sources for us to work with. However, we’ll have to download them before we can start playing around with them. In your terminal, now that you have imported NLTK into Python, enter:

nltk.download()

This will display the NLTK Downloader interface. It should look something like this:


At this point, type d and press enter. It will ask you the identifier of the package you want to download. We want to download the resources used in Natural Language Processing with Python, so we’ll type book and press Enter. Finally, to exit the downloader, type q and press Enter one more time.

Now, before we can actually do anything with these texts we’ll have to import them into Python. To do this, enter:

from nltk.book import *

That’s it!

You now have NLTK installed on your own VM in the cloud and can access it on any computer. Have fun learning about natural language processing with Python!
